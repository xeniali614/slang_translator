<H1>Slang Translator</h1>
Project 2 for COP3530

<h2>Description</h2>

The slang translator either:
- Takes in a word and searches for its definition within the dataset
- Lets you browse the dataset, sorted alphabetically, by upvotes, or by downvotes

This project is implemented using C++


- The dataset can be found here:
https://www.kaggle.com/datasets/therohk/urban-dictionary-words-dataset

<h2>Installing and Running the Project</h2>

1. Clone the project
2. Open project in IDE
3. Run main

<h2>Project Usage</h2>

1. Enter the menu option for searching, browsing, or exiting
2. Find definition of certain slang words or browse slang words
3. If browsing, see the time difference between the sorts!
